# 乐购商城 (LeGou Store) 项目说明

本项目是一个电商网站的前端模拟实现。项目包含用户管理、商品展示、商品详情浏览和购物车管理等核心功能。

## 功能模块与入口说明

### 1. 公共导航（全局功能）
* **逻辑控制**：`js/header-footer.js`
* **功能描述**：
    * 页面头部的“欢迎光临 乐购”和页脚的 Logo 图标点击后均跳转至 **首页** (`index.html`)。
    * 页面头部的“登录”和“注册”链接分别跳转至 **登录页** (`login.html`) 和 **注册页** (`register.html`)。

### 2. 网站首页
* **入口文件**：`index.html`
* **逻辑控制**：`js/index.js`
* **主要功能与跳转**：
    * **商品导航**：页面顶部和左侧包含商品分类导航。数据通过 `js/index.js` 读取 `data/nav.json` 和 `data/index.json` 动态生成。
    * **轮播图展示**：页面中心使用 Swiper 插件展示热门活动图片。
    * **楼层展示**：页面包含图书、服装、运动等不同品类的商品展示区域。
    * **商品跳转**：点击“每日推荐”区域的商品（如显示“限时抢”的卡片），会跳转至 **商品详情页** (`detail.html`)。
    * **购物车入口**：点击顶部搜索栏右侧或右侧悬浮栏中的“购物车”图标，均可跳转至 **购物车页** (`shoppingCar.html`)。

### 3. 用户注册
* **入口文件**：`register.html`
* **逻辑控制**：`js/register.js`
* **主要功能与跳转**：
    * **信息录入**：用户输入用户名、密码、手机号。
    * **验证码生成**：系统会自动生成随机字符验证码图片。
    * **表单验证**：系统会检查用户名唯一性、密码长度和手机号格式。
    * **注册成功跳转**：信息验证通过并点击“立即注册”后，系统弹出提示并自动跳转至 **登录页** (`login.html`)。
    * **已有账号跳转**：点击标题栏右侧的“登陆”链接，可直接跳转至 **登录页** (`login.html`)。

### 4. 用户登录
* **入口文件**：`login.html`
* **逻辑控制**：`js/login.js`
* **主要功能与跳转**：
    * **登录模式切换**：用户可以在“扫码登录”和“账户登录”之间切换。
    * **身份验证**：系统会比对输入信息与 `localStorage` 中的注册数据。
    * **登录成功跳转**：验证通过并点击“登录”后，系统弹出提示并自动跳转至 **首页** (`index.html`)。
    * **注册跳转**：点击登录框底部的“立即注册”按钮，会跳转至 **注册页** (`register.html`)。

### 5. 商品详情页
* **入口文件**：`detail.html`
* **逻辑控制**：`js/detail.js`
* **主要功能**：
    * **图片展示**：支持缩略图切换大图功能。
    * **规格选择**：用户可以选择商品的种类（如平装版、精装版）。
    * **数量调整**：用户可以增加或减少购买数量。
    * **评价展示**：页面底部展示商品的用户评价和评分统计。
    * **交互说明**：本页面的跳转主要依赖顶部的公共导航。

### 6. 购物车
* **入口文件**：`shoppingCar.html`
* **逻辑控制**：`js/shoppingCar.js`
* **主要功能**：
    * **商品列表**：展示用户添加的商品信息。
    * **金额计算**：系统会自动计算单件商品小计和订单总价。
    * **商品管理**：用户可以删除商品或调整商品数量。
    * **全选功能**：用户可以一键选中或取消选中所有商品。
    * **猜你喜欢**：点击底部的推荐商品，该商品会被直接添加到购物车列表中进行演示。
    * **交互说明**：本页面的跳转主要依赖顶部的公共导航。

## 目录结构说明

* `css/`: 存放所有样式文件。`reset.css` 用于重置浏览器默认样式。
* `js/`: 存放所有逻辑脚本。`header-footer.js` 处理公共头部和底部的逻辑。
* `imgs/`: 存放项目所需的图片资源。
* `data/`: 存放模拟用的 JSON 数据文件。
* `lib/`: 存放第三方库（如 Swiper）。
